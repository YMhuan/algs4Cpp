## Overview

algs4Cpp is a C++ library port of [algs4.jar](http://algs4.cs.princeton.edu/code/) with the goal to serve instructors and programmers who adopt the Algorithms, 4th Edition textbook and would prefer to follow and demonstrate the textâ€™s algorithms using C++.

## Documentation

The library was delivered as a [Visual Studio 2015](https://www.visualstudio.com) solution. The memory leak detector adopted is [Visual Leak Detector](https://vld.codeplex.com/).
